서버 규칙

즐겁고 공정한 게임 환경을 위해 다음 규칙을 모든 플레이어가 준수해야 합니다.

  • 게임 목표

    최후의 1팀이 승리

  • 팀 수

    4팀, 팀 당 5명

  • 목숨

    각 플레이어의 목숨은 3개(사람한테 죽었을 때, 목숨 0이 되면 관전자 모드)

  • PVP 규칙

    2주차 때부터 PVP 가능

  • 상점 시스템

    포인트 시스템 X, 좌표 0, 0에 상점(비밀 상인을 찾아보세요!!)

  • 게임 환경

    월드 보더(오버월드): 10,000 X 10,000

    월드 보더(네더): 625 X 625

    월드 보더(엔더): 5000 X 5000

    킵 인벤토리는 지원하지 않습니다.

    기본템:스테이크 64개 지급

  • 클라이언트 설정

    감마 조정 허용, 엑스레이(X-Ray) 사용 금지

규칙 위반 시 경고, 일시적 차단, 영구 차단 등의 조치가 취해질 수 있습니다. 관리자의 결정은 최종적이며, 항상 공정하게 적용됩니다.

서버 참여 방법

2025 SMP 서버에 참여하는 것은 매우 간단합니다. 아래 단계를 따라주세요:

  1. 마인크래프트 준비하기

    정품 마인크래프트 Java 에디션이 필요합니다. (버전 1.20.2)

  2. 게임 실행하기

    마인크래프트를 실행하고 멀티플레이 메뉴로 이동합니다.

  3. 서버 추가하기

    '서버 추가' 버튼을 클릭합니다.

  4. 서버 주소 입력하기

    서버 주소 2025smp.kro.kr을 입력합니다.

  5. 접속하기

    서버에 접속하여 즐겁게 플레이하세요!
